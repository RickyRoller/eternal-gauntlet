# Eternal Guantlet
This is my submission for the [Solana Speedrun 3 Game Jam](https://solanaspeedrun.com/).
Built with Bevy + Rust + WASM + Gameshift + Solana.

## Controls
- `WASD` for movement
- Left click to cast spell
